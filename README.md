unit-converter
===============

A small commandline unit converter.

Usage:
```
$ unit 100km/h
  27.78 m/s
  100 km/h
  62.14 mph
```
```
$ unit 1/2 week
  302400 seconds
  5040 minutes
  84 hours
  3.50 days
  0.50 weeks
  0.01 years
Unix time 0.5 weeks from now: 1435717504
```

With the following line in my bash rc: `alias unit="python3 path/unit-converter.py`
